# key-gin

> 基于go-gin框架生成激活key项目

- 此分支使用了`chromedp`


## 使用

### 到[releases](https://github.com/woytu/key-gin/releases)下载解压并运行

### 必须安装`Chrome`

```bash
# Windows
# 双击key-gin-windows.exe根据默认端口8000运行
# 或者在cmd、power shell中
key-gin-windows.exe


# Linux
nohup ./key-gin-linux -p 5000 >/dev/null 2>index.log &
```
