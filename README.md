# key-gin
基于go-gin框架生成激活key项目

## 使用
```bash
# 克隆项目
git clone https://github.com/woytu/key-gin.git
cd key-gin
# 然后到https://github.com/woytu/key-gin/releases下载自己系统的二进制运行程序

# Windows
# 双击key-gin-windows.exe根据默认端口8000运行
# 或者在cmd、power shell中
key-gin-windows.exe

# Linux
nohup ./key-gin-linux -p 5000 >/dev/null 2>index.log &
```
