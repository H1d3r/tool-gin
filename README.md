# tool-gin

> 基于go-gin框架生成激活key项目

- 此分支使用了`chromedp`


## 使用

### 到[releases](https://github.com/woytu/tool-gin/releases)下载解压并运行

### 必须安装`Chrome`

```bash
# Windows
# 双击tool-gin-windows.exe根据默认端口8000运行
# 或者在cmd、power shell中
tool-gin-windows.exe


# Linux
nohup ./tool-gin_linux_amd64 -p 5000 >/dev/null 2>&1 &
```
